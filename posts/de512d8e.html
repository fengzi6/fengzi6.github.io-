发表于2020-06-28|更新于2020-10-11|杂七杂八

支持 https 的二维码 api 服务并不多，这里是经过站长精心筛选后剩下的结果。

推荐日常使用搜狐视频提供的二维码 api 接口，稳定性很好，已使用多年。

使用方法很简单，替换 https://XXXX 为想要生成的文字或链接即可。
